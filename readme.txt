"Chicago Crime Analysis"


This program takes in data on Chicago crimes and areas and stores them into hashtables for easy lookup. After the data is stored, the user can enter a crime number and information about the crime is returned.

To run this program:

gcc main.c -o myProg
./myProg



Sample User Input:

Crimes.csv (Other crimes file from Data Files folder)
Areas.csv
Specific Crime #
